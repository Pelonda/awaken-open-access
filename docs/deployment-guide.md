# Deployment Guide

## Requirements
- Windows 10 / 11
- .NET Runtime
- SQL Server or SQL Express

## Server Deployment
1. Install SQL Server
2. Create database
3. Configure connection string
4. Launch Admin Server application

## Client Deployment
1. Install client application on each PC
2. Configure server connection
3. Lock screen activates automatically

## Database
- Tables are created automatically
- Default settings are seeded
- Supports large-scale deployments

## Notes
Production installers are available upon request for evaluation or pilot programs.
